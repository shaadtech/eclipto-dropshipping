# ECLIPTO Project Summary

## Project Overview
ECLIPTO is a dropshipping website built with Next.js that integrates with Amazon's product catalog and Supabase for authentication. The website features a modern, responsive design with comprehensive e-commerce functionality.

## Features Implemented

### Authentication System
- User registration and login functionality
- Secure authentication using Supabase
- Protected routes for authenticated users
- User profile management

### Amazon API Integration
- Simulated Amazon Product Advertising API integration
- Product catalog with categories and search
- Product details with specifications and related products
- Ready for connection to real Amazon Associates account

### User Interface
- Custom theme with unique color scheme
- Responsive design for all device sizes
- Modern UI components with animations
- Dark mode support

### Product Functionality
- Product listing with advanced filtering
- Category-based browsing
- Search functionality
- Product detail pages with tabs for information
- Related products recommendations

### Shopping Cart
- Add to cart functionality
- Quantity adjustment
- Order summary with tax calculation
- Checkout process

## Technical Implementation
- Next.js framework for server-side rendering and routing
- Tailwind CSS for styling
- TypeScript for type safety
- Supabase for authentication and user management
- Component-based architecture for maintainability

## Getting Started
To continue development:
1. Clone the repository
2. Install dependencies with `pnpm install`
3. Run the development server with `pnpm dev`
4. Access the site at http://localhost:3000
